# Dany Terrain V1

Base React/Vite de l'application terrain SOS Peinture & Réparation.

## Objectif

Interface simple pour Alban et l'équipe terrain :

- gros bouton Parler
- planning
- chantier du jour
- ouverture Waze
- rappels avant départ
- matériel prévu
- photos avant/après
- signalement problème

## Déploiement Netlify

Pour tester cette version :

- Branch: v1
- Build command: npm run build
- Publish directory: dist
